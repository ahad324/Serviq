# 📄 2. PRD (PRODUCT REQUIREMENTS DOC)

## 🎯 Goal

Build an AI-powered app that:

* Understands natural language requests
* Matches service providers
* Calculates pricing
* Simulates booking + lifecycle

---

## 👤 Target Users

* People needing local services (AC, plumber, etc.)

---

## ✅ Features

1. Natural language input (Urdu/English)
2. AI intent extraction
3. Provider matching (multi-factor)
4. AI reasoning display
5. Pricing breakdown
6. Booking confirmation
7. Service tracking
8. Feedback/dispute

---

## 📊 Success Criteria

* End-to-end working flow
* Clear AI reasoning
* Clean UI
* Handles edge cases
